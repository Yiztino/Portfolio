Acerca de mí

También conocido por mi nombre real Luis Lemus, soy un desarrollador de software interactivo y videojuegos que le apasiona hacer videojuegos y código que pueda transmitir mis ideas o pueda ser útil para alguien. Algunos motores de creación de juegos que disfruto usar son Unity, Unreal Engine y Godot. Sé programar en C##, C++, lua, javascript, php y html. También uso markdown. :D
